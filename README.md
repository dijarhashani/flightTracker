A modern, lightweight, pure HTML/CSS/JavaScript frontend for the **AirBook** Flight Tracking & Booking System.  
This UI connects directly to the AirBook Go backend API and provides users with:

- Airport search  
- Live flight lookup  
- User authentication (Login/Register)  
- Favorite airports  
- Flight bookings  
- Personal dashboard  
- Real-time plane map lookup  
- Responsive design  


---

## Features

### Authentication
- Register new users  
- Login with secure cookie handling  
- Automatic session check (`/auth/me`)  
- Logout  

### Airport & Flight Search
- Search flights by airport  
- Search airport status  
- Fetch flight & plane details from backend APIs  
- Live map lookup by HEX code

### Favorites System
- Save/unsave favorite airports  
- View saved favorites  
- UI updates instantly

### Bookings
- Book flights  
- View personal bookings  
- Remove bookings  
- Clean UI layout with Bootstrap

### Responsive UI
- Clean CSS  
- Mobile-friendly  
- Bootstrap-based components  
- Custom animations and loaders
